# Windows 清理工具

一个全面的批处理脚本，用于清理 Windows XP、Vista、7、8、10 和 11 系统中的临时文件、缓存和垃圾数据。

## 功能特点

### 核心清理功能
- **临时文件**：清理用户和系统临时文件夹
- **系统缓存**：删除 Windows 更新缓存、预读取文件和日志
- **浏览器缓存**：支持 Internet Explorer、Chrome、Edge 和 Firefox
- **回收站**：清空所有驱动器的回收站
- **最近文件**：清除最近文件和缩略图缓存
- **第三方软件**：清理热门应用程序的缓存

### 安全特性
- **管理员检查**：需要管理员权限
- **错误抑制**：防止错误消息中断清理过程
- **条件执行**：仅处理存在的文件夹
- **非破坏性**：仅删除安全的、可重建的数据

### 兼容性
- Windows XP
- Windows Vista
- Windows 7
- Windows 8 / 8.1
- Windows 10
- Windows 11

## 使用方法

### 前提条件
- 需要管理员权限
- 为获得最佳效果，请关闭所有正在运行的应用程序

### 运行脚本

1. 右键点击 `Windows_Cleaner_Enhanced.bat`
2. 选择"以管理员身份运行"
3. 确认清理提示
4. 等待清理完成
5. 如有需要，重启计算机

## 清理部分

脚本分为 6 个主要部分：

1. **临时文件** - 用户和系统临时文件夹
2. **系统缓存** - Windows 更新、预读取、CBS 日志
3. **浏览器缓存** - IE、Chrome、Edge、Firefox
4. **Windows 系统** - 安装程序缓存、性能日志
5. **回收站** - 所有驱动器
6. **最近文件** - 缩略图、图标缓存

以及可选的第三方软件清理

- 不会触及系统关键文件
